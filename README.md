# To install

*  `composer install`

#  Prep
*  `composer selenium-server`

#  To run local version
*  `composer behat`

#  To run on CrossBrowserTesting
*  You will need an account from CBT
*  First, update `behat-cbt.yml`
   *  update user w/ user account from CBT
   *  update key w/ key from CBT
*  `composer behat-cbt`
